# DH KM 助理正式部署

## Render 方案

1. 把当前项目上传到 GitHub、GitLab 或 Bitbucket 仓库。
2. 登录 Render 后，新建 Blueprint 或 Web Service。
3. 如果使用 Blueprint，直接选择仓库根目录下的 `render.yaml`。
4. 如果使用普通 Web Service，选择 Docker 运行时，直接读取仓库里的 `Dockerfile`。
5. 部署完成后，Render 会先分配一个固定的 `onrender.com` 地址。
6. 如果你有自己的域名，再到 Render 的 Custom Domains 里绑定域名并按提示配置 DNS。

## 当前项目已准备好的文件

- `Dockerfile`
- `requirements.txt`
- `render.yaml`

## 需要你自己准备的东西

- 一个代码托管仓库
- 一个 Render 账号
- 如果要自己的固定域名，还需要一个域名和 DNS 控制权
